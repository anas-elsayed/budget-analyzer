# Budget Analyzer V1

## Features
- Contacts import
- Transaction import
- Tutoring categorization
- Faculty salary detection
- Dashboard analytics
- Chart.js visualization
- GitHub Pages compatible
- Supabase ready

## Setup

1. Upload all files to GitHub repository
2. Enable GitHub Pages
3. Open js/supabase.js
4. Paste your Supabase URL and anon key
5. Commit changes

## Supabase Tables

Run the SQL provided earlier in Supabase SQL Editor.
